# Google App Launcher

Simple access to Google Applications with a tool bar button

> [Get on AMO](https://addons.mozilla.org/en-US/firefox/addon/google-app-launcher/)

Currently has a hard coded links to popular Google applications. There is no customization support.

Supported apps:


* `gplus`: Google+
* `gmail`: Gmail
* `gcal`: Calendar
* `gdoc`: Docs
* `gsheets`: Sheets
* `gdrv`: Drive
* `gphotos`: Photos
* `gmaps`: Maps
* `gplay`: Play
* `gytube`: YouTube
* `gnews`: News
* `gwall`: Wallet
* `ghang`: Hangouts
* `gcont`: Contacts
* `gtrans`: Translate
* `gkeep`: Keep
* `gmusic`: Music
* `google`: Search
