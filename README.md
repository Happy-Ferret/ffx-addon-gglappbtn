# Google App Launcher

Simple access to Google Applications with a tool bar button

Currently has a hard coded links to popular Google applications. There is no customization support.

Supported apps:

* gplus: Google+
* gmail: Gmail
* gcal: Calendar
* gdoc: Docs
* gdrv: Drive
* gphotos: Photos
* gmaps: Maps
* gplay: Play
* gytube: YouTube
* gsheets: Sheets
* gnews: News
